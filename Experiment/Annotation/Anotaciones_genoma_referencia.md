# Anotaciones genoma *Arabidopsis thaliana*

## Fichero en formato gff3 con la anotación de la localización de los genes, transcritos y exones sobre el genoma de referencia, en este caso, de *Arabidopsis thaliana*.

Para ello, dentro de **Ensembl** podemos encontrar disponible para descagar los "*Gene sets*" para el genoma de referencia. 
Aparecen disponibls distintos archivos, con las anotaciones de cada cromosoma por separado, y otro archivo con las anotaciones de todo el genoma completo.

**Arabidopsis_thaliana.TAIR10.62.gff3.gz**  #Es el archivo que deberíamos descargar para obtener las anotaciones de todo el genoma
