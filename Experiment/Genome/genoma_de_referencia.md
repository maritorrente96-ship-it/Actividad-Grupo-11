# Genoma de referencia en formato FASTA
Para la realización del análisis que hemos elegido se usa la planta
*Arabidopsis Thaliana* como organismo modelo. Por ello, en esta carpeta
se encontraría el fichero FASTA con el genoma completo de dicha planta. 
Dicho fichero FASTA con el genoma nos lo podemos descargar de la base de 
datos plants.ensembl.org
